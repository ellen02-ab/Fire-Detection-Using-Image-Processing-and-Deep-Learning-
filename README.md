Fire Detection Using Image Processing and Deep Learning
📌 Project Overview

This project focuses on detecting fire in images using image processing and deep learning techniques. The system classifies images into two categories:

  #Fire
  #Non-Fire

The model is trained on a dataset containing fire and non-fire images and uses machine learning and deep learning algorithms to accurately detect fire from images.
🛠 Technologies Used

Python
TensorFlow / Keras
OpenCV
Scikit-learn
NumPy
Pandas
Matplotlib
Seaborn

🔄 Project Workflow

1️⃣ Dataset Collection
2️⃣ Image Preprocessing
3️⃣ Data Cleaning
4️⃣ Image Augmentation
5️⃣ Train-Test Split
6️⃣ Model Training
7️⃣ Model Evaluation
8️⃣ Fire / Non-Fire Prediction

📌 Conclusion

This project successfully demonstrates the use of image processing and deep learning techniques to detect fire in images. By training the model on a dataset containing fire and non-fire images, the system is able to classify images and identify the presence of fire with good accuracy.
